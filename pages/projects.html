Game Development

Tower Defense - Python

This is my most recent game development project. My friend and I are working together on re-creating our own version of the classic Tower Defense game. The idea of the game is to build towers of different types to defend against incoming enemies.

As of right now, we have different classes of towers with different damage and range. I am currently working on the enemy pathing to allow the game to read the path from a file instead of hard coding the path which is currently the case.

Galactic Defender - Python

This is my second game using the Pygame Python library; it is a re-make of the classic space invaders game. However, my version includes many new components including boss ships, power-ups and different types of enemies.

This is the first game that I independently create without heavy reliance on sources (except official documentation), and I am proud to say that this is my first fully functional game that I have implemented
