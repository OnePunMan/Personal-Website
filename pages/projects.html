RememberMeThis - Android App

RememberMeThis is an educational android app that allows students to remember course material in the most effective way via the Spaced Repetition strategy. Spaced repetition is a method in which the content are reviewed in gradually increasing intervals. This has proven to maximize the amount of knowledge retained in memory.

The idea of this app was inspired by the language learning website Memrise, which follows a similar protocol in which users are prompted to review words learned at regular, and increasing time intervals. I used this website to learn over 1500 new French words during the time in Montr√©al and I have found it to be extremely effective.

I created this app that follows a similar style as Memrise, but I generalized it so that it allows custom inputs from users. This means that students can use this app to add their course definitions, theorems and questions. I am currently using this app during my school term to test out its effectiveness and I will be continuing to work on this project. This is an open source repo and the link can be found here.

PicFive! - Web App

PicFive is the project that my team and I worked on at the UofTHacks 2017 Hackathon. The game prompts the player with five words (nouns) and the player has to upload an image that closely associates with as many of the five words as possible.

Through the use of the Clarifai API, we are able to use image processing and extract words associated with elements in the images. With this data, the game will return an adjusted score between 1 and 100 indicating how well of a match the photo is with respect to the five words. In addition, the top 10 scores are displayed on the Leaderboards.

Integral Visualizer - Desmos

This is a simple tool I made using Desmos graphing calculator that can be used to easily visualize definite integrals with colours representing their signed areas. The input function is customizable and the bounds can be changed to any values that form a valid integral.

Graph Art: Minion - Desmos

This is a minion from the movie Despicable Me created using the Desmos Calculator. I used different mathematical functions with restricted domains and ranges to create the outline of the image. I used inequalities to represent the coloured regions on the image.

WatRGoose - Android Mini-game

Being University of Waterloo's best well-known signature animal - the Canadian Goose, I decided to make a mini game that is related to them. Together with two of my friends, we participated in MLH's LocalHackDay 2016 event for our first 12-hour mini Hackathon. This is our first time experimenting with Android App development. Using Android Studio as our framework, we have learned (and struggled) a lot during the development of this project.

This project is essentially a mini-game where you have to flip 9 cards to the right orientation to reveal a picture of a beautiful goose. The catch is, however, that when a card is flipped, all its adjacent cards will flip too, which means you could end up messing up an already partially solved puzzle!

Pong - Python

Ah, if it isn't the class Pong. This may sound like an easy game, but there are a few tricky components in this project that are not present in others. The physics of the ball is one of them. I have implemented the game in a way such that the speed of the ball can be changed depending on the speed of the paddle and its friction coefficient.

Also, one of the harder components of the game is the AI, I am currently working on a Pong AI that will be able to effectively play against the player. That's what makes this game challenging to code.
